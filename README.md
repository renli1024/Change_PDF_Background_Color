# Change_PDF_Background_Color
使用Java iText7库，将PDF背景色改为豆沙绿（护眼色），白底论文看着太累眼了orz。

## 使用方法
1. 下载导入iText7 jar包，修改PageBackgrounds.java文件中的相应路径名称。
2. 直接运行PageBackgrounds.java文件即可。
